# Debugging

## Enable Debug Mode

> **Default**: `debug=false`

Java SDK logging uses SLF4J. Configure your own logging settings. Example:

```xml
<!-- enable debug logs -->
<logger name="com.volcengine.sdkcore" level="debug"/>
```
